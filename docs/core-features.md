# 核心功能

本文件描述了 "墨阁" 应用当前已实现的核心功能。

---

## 核心功能现状

应用目前已具备稳定可靠的核心功能：

1.  **用户系统**:
    - 支持账号密码注册与登录。
    - 支持 GitLab 第三方授权登录。
    - 提供个人中心，可修改个人资料与密码。

2.  **设定集管理**:
    - **项目管理**: 支持小说项目的创建、编辑、删除和列表展示。
    - **设定分类**: 提供四大设定分类（角色、系统、世界、辅助）的完整管理功能。
    - **设定 CRUD**: 支持所有类型设定的创建、编辑、查看和删除操作。
    - **设定库**: 提供独立的设定库，支持跨项目的设定复用。
    - **筛选排序**: 支持多维度筛选（名称、标签、类型等）与排序。
    - **视图切换**: 支持列表视图和卡片视图两种展示模式。
    - **关联管理**: 项目可关联多个设定，设定删除时检查关联关系。

3.  **大纲管理**:
    - 支持大纲的创建、编辑、删除和列表展示。
    - 支持多维度筛选（名称、类型、时代、标签等）与排序。
    - 具备核心的 **AI 流式生成** 大纲内容的能力。
    - 提供基于 Markdown 的结构化内容查看与编辑功能。

4.  **基础体验**:
    - 支持明/暗两种主题模式。
    - 支持中/英双语切换。
